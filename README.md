# Bear API Service

Thư viện tham chiếu 1 số API public tại https://api.nguyeanhung.com/

## Cài đặt & Sử dụng

### Cài đặt thư viện

Cài đặt thư viện thông qua composer với lệnh sau

```shell
composer require bear8421/bear-service
```
